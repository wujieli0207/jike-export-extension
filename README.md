# 即刻导出

即刻导出是一个浏览器插件，可以导出即刻动态为本地文件，[下载地址](https://jike-export.wujieli.com/)

支持导出为 Markdown、纯文本、excel、csv 格式的文件

1. 支持导出自己和即友的动态列表，同时支持导出个人收藏
2. 支持导出图片、动态链接、引用动态等完整信息
3. 支持无缝导入 Obsidian、Heptabase 等笔记工具
